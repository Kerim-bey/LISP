Условие: Определите функцию, осуществляющую удаление указанного количества последних элементов исходного списка.

Код: 
``` 
(defun len (list)

    (cond ((null list) 0)

        (t (+ 1 (len (cdr list))))

    )

)

(defun del (lst m)
    (cond ((null lst) 0)
    ((eql (len lst) m) nil)
    (t (cons (car lst) (del (cdr lst) m)))))
```
